AUDIO SYNC PRO
----------------------------------------

Audio Sync Pro is an Adobe After Effects script designed to simplify
audio-driven animation workflows.

It allows you to quickly extract audio amplitude data and apply it to
animation properties using expressions.


FEATURES
----------------------------------------
- One-click audio extraction
- Keeps only "Both Channels" for cleaner control
- Quick syncing of animation properties to audio
- Adjustable intensity control
- Simple and lightweight UI panel


HOW TO USE
----------------------------------------

1. EXTRACT AUDIO
   - Select a single audio layer in your composition
   - Click "Extract"
   - The script will create an "Audio Amplitude" layer
     with only the "Both Channels" data

2. SYNC ANIMATION
   - Select a property (Scale, Rotation, Opacity, etc.)
   - Set your desired Intensity value
   - Click "Sync"
   - The property will now react to the audio


REQUIREMENTS
----------------------------------------
- Adobe After Effects
- Expression-enabled properties


INSTALLATION
----------------------------------------
1. Copy the .jsxbin file to:
   Adobe After Effects > Support Files > Scripts > ScriptUI Panels

2. Restart After Effects

3. Open from:
   Window > Audio Sync Pro


NOTES
----------------------------------------
- You must select exactly one audio layer before extracting
- You must select at least one property before syncing
- Intensity must be a non-zero numeric value
